# 🛒 E-Commerce Microservices Project

This is a backend-based E-Commerce system built using a Microservices architecture with .NET. The project is designed to demonstrate scalable system design, separation of concerns, and real-world backend development practices.

---

## 🚀 Technologies Used

- ASP.NET Core Web API
- C#
- Microservices Architecture
- SQL Server
- Entity Framework Core
- Docker (if used)
- RESTful APIs
- Git & GitHub

---

## 🏗️ Architecture Overview

The project is divided into multiple independent services, each responsible for a specific business domain, such as:

- Product Service
- Order Service
- User Service
- Payment Service (if applicable)
- API Gateway (if applicable)

Each service is designed to be loosely coupled and independently deployable.

---

## 📌 Key Features

- Scalable Microservices architecture
- Clean separation between services
- RESTful communication between services
- Database per service approach
- Modular and maintainable code structure
